# \#GoWeek

Repositório com sistemas web e mobile que listam, criam e permitem dar likes em tweets com atualizações em tempo real.

---

## Sistema backend (API)

### Stack:

 - Node
 - Express
 - MongoDb

### Bibliotecas usadas:

 - Mongoose
 - Socket.io
 - Cors
 - Nodemon

## Sistema frontend

### Stack:

 - React

### Bibliotecas usadas:

 - Axios
 - React-router-dom
 - Socket.io-client
